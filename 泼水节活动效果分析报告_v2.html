Governing Thought

泼水节在 50 個 LMT/Event 限時賽單中以 3.23% 參與率排名第一，是同類均值（1.49%）的 2.17 倍，且在無獨立付費激勵、無專屬賞金的前提下達成——這說明節日 IP 驅動的玩法吸引力足以獨立拉動流量；單局僅 2.89 分鐘的極速節奏使中退率降至 4.01%（比 LMT 均值低 71%），接近 Core MP 日常水準，確認局內體驗完整；賽單七留 6.97% 高於同類均值（6.41%），在 LMT 中排名 #6，代表有效的 7 日粘性轉化。核心結論：泼水节的活躍拉動效果不依賴外部獎勵，其「低門檻 + 高完整度 + 極速回路」三重特徵是可複製的節日活動設計範式。

Supporting Argument 1：參與率同類第一，無需付費驅動
SA1 — 玩家参与率
泼水节參與率 3.23%，排名 #1/50，是同類均值的 2.17 倍
Fact

泼水节平均玩家參與率 3.23%，在全部 50 個 LMT/Event 賽單中排名第一（#1/50）。同類均值僅 1.49%，排名第二的 Featured-Khandor Hideout 24/7 為 2.74%——泼水节領先第二名 0.49pp（+18%）。值得注意的是，泼水节的 3.23% 幾乎等同於 Core MP 日常均值（3.07%），代表一個 限時節日活動在無付費激勵下能複製日常賽道的流量吸引力。

Implication

節日 IP（潑水節主題）本身即為有效流量觸發點，玩家進入無需外部利益驅動。這區別於多數 Featured 賽單依賴道具獎勵或獨特機制吸引偶發參與的邏輯——泼水节的流量基礎更穩固，參與動機來自玩法本身的節慶體驗感。

Action

建議以泼水节作為節日 IP 活動設計的參考基線：在無強制付費綁定的前提下，若一個限時賽單能達成 ≥2.5% 參與率，即可視為流量效果達標標準。

Supporting Argument 2：場均擊殺非優勢維度，但符合極速節奏定位
SA2 — 場均擊殺 × 單局時長
21.84 擊殺看似低於 Core MP，但本質是 2.89 分鐘極速節奏的必然結果
Fact

泼水节場均擊殺 21.84，與 Core MP 均值（22.65）基本持平，遠低於 LMT/Event 均值（75.67）。但泼水节單局時長僅 2.89 分鐘，是 Core MP 均值（4.13 分鐘）的 70%、LMT 均值（10.23 分鐘）的 28%。若以「每分鐘擊殺數」換算，泼水节約 7.55 殺/分鐘，高於 Core MP 的 5.48 殺/分鐘（+38%），是真正的高強度極速局。

Implication

以「場均擊殺絕對值」評估泼水节是錯誤框架。其設計邏輯是透過極短局長壓縮時間成本、提高單位時間戰鬥密度，從而降低輕度玩家的參與門檻——這正是其參與率冠軍的底層機制。LMT 均值場均擊殺高（75.67）的賽單反而多為長局或特殊玩法，中退率也更高（13.93%）。

Action

在後續節日活動設計評估中，對極速局類型應以「每分鐘擊殺密度」而非「場均擊殺總量」作為玩法強度評估指標。

泼水节落於「極速高密度」象限，與 Core MP / Competitive MP 均值為鄰。LMT 均值雖然場均擊殺最高，但單局超過 10 分鐘，中退率也達 13.93%。泼水节以最短的時長實現與 Core MP 持平的密度，是其低門檻高完整度的設計優勢。
Supporting Argument 3：中退率接近日常賽道，局內體驗完整度超越同類
SA3 — 中退率
中退率 4.01%，比 LMT/Event 均值低 71%，與 Core MP 日常水準齊平
Fact

泼水节中退率 4.01%，LMT/Event 均值為 13.93%，BR Course 均值 5.08%，Core MP 均值 3.01%。泼水节的中退率不僅比同類活動低 71%，甚至與 Core MP（最低類別之一）的差距僅 1pp——在一個限時節日模式中實現日常賽道級別的完局率，是強力的玩法信心指標。

Implication

許多 LMT/Event 賽單存在「吸引玩家進局、但玩一半就退」的體驗問題（均值 13.93%），這代表玩法新鮮感難以支撐完整局面。泼水节突破了這個模式：玩家進局後選擇打完的比例接近日常習慣賽道。極短單局（2.89 分鐘）本身亦有助於降低中退動機——局太短導致「退不退都差不多」，從而自然拉高完局率。

Action

中退率 ≤5% 可作為「限時活動玩法品質及格線」。後續活動上線前，建議以 Beta/軟上線期的中退率作為快速健康指標，發現 >8% 即啟動玩法調整。

LMT/Event 類別中多數賽單中退率介於 8–20%，而泼水节 4.01% 不僅是 LMT 類別的最優值，亦接近 Core MP 日常（3.01%），確認局內玩法的高完整度。
Supporting Argument 4：賽單七留超均值，節日 IP 有效轉化短期流量為 7 日粘性
SA4 — 赛单七留
七留 6.97% 高於 LMT 均值（6.41%），在 50 個限時賽單中排名 #6
Fact

泼水节賽單七留 6.97%，LMT/Event 均值為 6.41%（泼水节超出 +8.7%）。在 50 個 LMT/Event 賽單的七留排名中，泼水节位列第六——前五名依序為：Sudal Harbor 10v10（10.67%）、Core-Train（10.17%）、Train（10.17%）、Featured-Vacant（7.45%）、裝卸貨場-新春（7.30%）。次日留存 72.1%，略低於 LMT 均值 85.1%，差距可部分歸因於活動窗口的「首日高峰後自然衰退」效應。

Implication

七留超均值代表泼水节不只是「打了就走」的流量刺激，而是能讓玩家在 7 日後仍有一定比例繼續登入。對於一個無付費綁定、短期上線的節日模式，七留跑贏均值是有效粘性轉化的具體證明。次日留存偏低（72.1% vs 均值 85.1%）反映活動初期的高強度曝光後自然降溫，屬正常活動衰減曲線。

Action

建議未來節日活動在泼水节基礎上測試「7 日期間每日任務」機制，以進一步拉升次日留存（目標 ≥80%），同時維持七留優勢。
